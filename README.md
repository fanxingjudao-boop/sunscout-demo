# SUNSCOUT — デモサイト

太陽光発電所の衛星植生モニタリング（SUNSCOUT）のデモ用ホームページです。

- `index.html` — ランディングページ（実測NDVIデータ内蔵・外部リソース非依存）
- `demo.html` — 月次レポートのフルデモ画面（地図表示に地理院タイルを実行時読込）

## データ出典

- Copernicus Sentinel-2 L2A — Contains modified Copernicus Sentinel data (2024–2025)
- [国土地理院（地理院タイル・ジオコーディング）](https://maps.gsi.go.jp/development/ichiran.html)
- FIT事業計画認定情報（経済産業省）

事業者名はサンプル表記です。数値（敷地輪郭・面積・NDVI時系列・欠測月）は実測値です。
損失試算は優先度判断用の概算であり、金額の確定は現地実測後に行います。
